# Contributing

Thanks for contributing. Keep changes small and focused.

By participating, you agree to follow the Code of Conduct:
CODE_OF_CONDUCT.md

## How to Contribute
- Open an issue for niche requests or data source ideas.
- Keep edits readable and avoid large rewrites without discussion.
- Do not add private data or sensitive personal data.
- Use the pull request template when submitting changes.

## Security Issues
Security concerns should follow the Security Policy:
SECURITY.md

## What Helps Most
- New registry or association sources.
- Edge cases and failure modes.
- Improvements to clarity or usability of the workflow.
